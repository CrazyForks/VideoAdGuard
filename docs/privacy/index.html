隐私政策

VideoAdGuard 致力于保护您的隐私和个人信息安全

最后更新：2025年8月15日

1. 概述

VideoAdGuard（以下简称"本扩展"或"我们"）是一个基于大语言模型的B站视频植入广告检测工具。我们非常重视您的隐私权，本隐私政策详细说明了我们如何收集、使用、存储和保护您的个人信息。

隐私承诺

我们郑重承诺：VideoAdGuard 不会收集您的个人身份信息，不会跟踪您的浏览行为，不会向第三方出售您的任何数据。

2. 信息收集

2.1 我们收集的信息

为了提供广告检测服务，本扩展可能会收集以下信息：

  • 视频字幕内容：仅用于分析是否包含植入广告，分析完成后不会永久存储
  • 视频音频片段：在用户启用音频识别功能时，临时提取音频用于转录分析
  • 视频基本信息：如视频ID（BV号）、标题等，用于缓存检测结果
  • 用户配置信息：您在扩展设置中配置的API密钥、模型选择等

2.2 我们不收集的信息

  • 您的个人身份信息（姓名、邮箱、电话等）
  • 您的浏览历史记录
  • 您的搜索记录
  • 您的账户登录信息
  • 除广告检测外的其他视频内容

3. 权限说明

本扩展申请的浏览器权限及其用途：

权限类型 具体权限 使用目的
存储权限 storage 保存用户设置、缓存检测结果
标签页权限 tabs 获取当前标签页信息，与内容脚本通信
活动标签页权限 activeTab 仅在用户主动操作时访问当前页面
哔哩哔哩域名权限 https://*.bilibili.com/* 在B站页面运行广告检测功能
Groq API权限 https://api.groq.com/* 调用音频转录服务（用户自选功能）

4. 信息使用

我们收集的信息仅用于以下目的：

  1. 广告检测服务：分析视频内容以识别植入广告
  2. 功能优化：改进检测准确率和用户体验
  3. 缓存机制：本地缓存检测结果以提升性能
  4. 技术支持：排查和解决技术问题

数据处理原则

  • 最小化原则：仅收集必要的信息
  • 透明化原则：明确告知数据用途
  • 目的限制原则：不用于其他目的
  • 安全保护原则：采取技术措施保护数据

5. 数据存储与安全

5.1 本地存储

  • 用户设置：存储在浏览器本地，不上传到服务器
  • 检测缓存：本地存储24小时，自动过期删除
  • API密钥：加密存储在本地，不会传输给我们

5.2 第三方服务

当您选择使用音频识别功能时，音频数据会发送至以下服务商：

  • Groq：用于语音转录服务，符合其隐私政策
  • 用户自定义AI服务：根据您配置的API endpoint

5.3 安全措施

  • 采用HTTPS加密传输
  • API密钥本地加密存储
  • 定期清理临时数据
  • 不记录用户操作日志

6. 数据共享与披露

我们承诺：

  • 不会出售：绝不向任何第三方出售您的个人信息
  • 不会共享：不会与广告商、营销公司共享数据
  • 有限披露：仅在法律要求的情况下才可能披露信息
  • 匿名化处理：如需分析统计，会完全匿名化处理

特殊情况

仅在以下情况下，我们可能会披露您的信息：

  • 法律法规要求
  • 司法机关调查
  • 保护用户和公众安全
  • 获得您的明确授权

7. 用户权利

您拥有以下权利：

  • 知情权：了解我们如何处理您的信息
  • 访问权：查看我们存储的您的信息
  • 更正权：纠正不准确的信息
  • 删除权：要求删除您的个人信息
  • 停用权：随时停止使用本扩展
  • 数据迁移权：导出您的设置数据

如何行使您的权利：

  1. 通过浏览器扩展管理页面卸载扩展
  2. 清除浏览器存储数据
  3. 联系我们的支持团队

8. 数据保留

数据类型 保留期限 删除方式
检测缓存 24小时 自动过期删除
用户设置 直至卸载扩展 卸载时自动清除
临时分析数据 处理完成即删除 自动删除
错误日志 不收集不存储 无需删除

9. 第三方服务

本扩展可能使用以下第三方服务：

  • AI模型服务商：如OpenAI、智谱AI、DeepSeek等，用于文本分析
  • Groq：用于音频转录服务
  • GitHub：用于开源代码托管和更新

这些第三方服务有各自的隐私政策，我们建议您仔细阅读。我们不对第三方服务的隐私做法负责。

10. 跨境数据传输

由于使用国际AI服务，您的数据可能会传输到其他国家或地区。我们会：

  • 仅使用符合数据保护标准的服务商
  • 确保传输过程的安全性
  • 遵守相关法律法规要求
  • 在可能的情况下优先选择本地服务商

11. 未成年人保护

我们非常重视未成年人的隐私保护：

  • 本扩展不专门面向未成年人设计
  • 不会故意收集未成年人的个人信息
  • 如发现收集了未成年人信息，会立即删除
  • 建议未成年人在监护人指导下使用

12. 隐私政策变更

我们可能会不时更新本隐私政策，以反映我们做法的变化或法律要求的变更。重大变更时，我们将：

  • 在扩展中显示通知
  • 在官方网站发布公告
  • 通过GitHub仓库通知
  • 给予用户适当的过渡期

继续使用本扩展即表示您接受更新后的隐私政策。
